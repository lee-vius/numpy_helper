numpy.array

numpy.array(object, dtype=None, copy=True, order='K', subok=False, ndmin=0) Link to Documentation

Create an array 

     Parameters:
               object:  array_like
               dtype: data-type, optional
               copy: bool, optional
               order: {'K','A','C','F'} , optional
               subok: bool, optional
               ndmin: int, optional
     Returns:
               out: ndarray

Example:

Upcasting: 

     >>> np.array([1, 2, 3.0])
     array([ 1.,  2.,  3.])

Multiple dimension: 

     >>> np.array([[1, 2], [3, 4]])
     array([[1, 2],
            [3, 4]])

Minimum dimensions 2: 

     >>> np.array([1, 2, 3], ndmin=2)
     array([[1, 2, 3]])

Type provided: 

     >>> np.array([1, 2, 3], dtype=complex)
     array([ 1.+0.j,  2.+0.j,  3.+0.j])
