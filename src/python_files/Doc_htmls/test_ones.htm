numpy.ones

numpy.ones(shape, dtype=None, order='C') Link to Documentation

Return a new array of given shape and type, filled with ones. 

     Parameters:
               shape: int or sequence of ints
               dtype: data-type, optional
               order: {'C', 'F'}, optional, default: C
     Returns:
               out: ndarray

Example: 

     >>> np.ones(5)
     array([1., 1., 1., 1., 1.])
               

     >>> np.ones((5,), dtype=int)
     array([1, 1, 1, 1, 1])
               

     >>> np.ones((2, 1))
     array([[1.],
            [1.]])
